# OshiPushle

**OshiPushle** is a daily guessing game (Wordle-style) based on the collectible card game **Oshi Push!**.

Test yourself each day by guessing the card of the day using its attributes: Color, Type, Level, Power, Life, and Cost.

## Features

-   **Daily Mode**: One new card to guess every 24 hours.
-   **Endless Mode**: Keep guessing cards in a row as long as you like.
-   **Platform Mode**: Filter cards by a specific platform.
-   **Fully Responsive**: Play comfortably on mobile or desktop.


## Technologies

-   **HTML5 / CSS3** (with locally-hosted fonts: Inter, Outfit, JetBrains Mono)
-   **Vanilla JavaScript**
-   **Cloudinary**

## Local Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/your-username/oshicards.git
    ```
2.  Change directory into the game folder:
    ```bash
    cd oshipushle
    ```
3.  Open `index.html` in your browser or use a local server such as Live Server.
